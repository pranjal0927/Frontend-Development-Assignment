To run the application you need to install below libraries and make sure you have a strong internet to downlaod

1.Install node_modules
    npm install

2. Install Tailwindcss 
    npm install tailwindcss@latest postcss@latest autoprefixer@latest

3. Create tailwindcss config file
    npx tailwindcss init

4. Install react - redux-toolkit
   npm install @reduxjs/toolkit react-redux react-router-dom
  
5. Install React-router-dom 
  npm install react-router-dom


To run the app please use npm start
